# My Web Development Journey
A place where I save my coding experiments as I learn full-stack web development.

## What's inside?

### 1. Flexbox and Grid (`/FlexGrid` folder)
This folder shows how to arrange items neatly on a webpage using two powerful tools: Flexbox and Grid. Think of them as invisible tables that keep your content organized.
* **Flexbox (Rows & Columns)**: How to place items side-by-side, space them out evenly, or perfectly center a box right in the middle of the screen.
* **Grid (Website Layouts)**: How to divide a webpage into simple chunks, like creating a top title bar, a side menu, and a large main reading area.

### 2. CSS Positions (`/Position` folder)
This folder is a very simple, visual guide to understanding how elements "move" and "stick" around the screen. I purposely wrote the code to be incredibly simple so anyone can read it!
* **Static**: This is how elements on a website naturally stack on top of each other, one by one.
* **Relative**: Nudging a box slightly left, right, up, or down from where it is supposed to sit.
* **Absolute**: Breaking a box completely out of the rules and pinning it exactly wherever you want inside its parent container (like pinning it to the bottom right corner).
* **Fixed**: Gluing a box to the browser window so it never moves, even when you scroll the webpage up and down!
* **Sticky**: A magical box that acts normal when you first scroll, but "sticks" to the top once it hits the edge of your screen, eventually letting go when its container ends.
